## How to Download
1. Go to the [Sphire Executor Releases](https://github.com/deplane-d/saphire-executor/releases) page.
2. Download the latest version suitable for your OS.
3. Extract the contents and navigate to the extracted folder.

## How to Use
Launch Sphire Executor and follow these steps:
1. Select a script to execute.
2. Configure any necessary parameters.
3. Click the 'Run' button to start the execution.

## Main Buttons
- **Run**: Starts the selected script.
- **Stop**: Terminates the running script.
- **Reset**: Clears all configurations and selections.

## Beginner Scripts
- **Basic Script**: Execute a simple command and retrieve the output.
- **Loop Script**: Demonstrate basic looping functionality.

## Adding Custom Scripts
To add custom scripts:
1. Navigate to the 'Scripts' folder within the Sphire Executor directory.
2. Create a new script file with the desired functionality.
3. Ensure the script adheres to Sphire Executor's scripting standards.

## Possible Issues
- **Script Fails to Execute**: Check script syntax and parameters.
- **Application Crashes**: Ensure system requirements are met and restart the application.

## Safety Warnings
- Always review scripts from unknown sources before execution.
- Avoid executing scripts with destructive commands or operations.

Как пользоваться? 🚀

При первом запуске Sphire Executor интерфейс может показаться сложным — особенно если вы новичок. Не переживайте, вот краткое объяснение:

Основные кнопки

INJECT — Подключает executor к Roblox
EXECUTE — Выполняет скрипт
CLEAR — Очищает редактор
OPEN — Открывает файл скрипта (.txt или .lua)
SAVE — Сохраняет текущий скрипт

Дополнительные кнопки

HELP — Показывает это руководство
SUPPORT — Открывает ссылки на поддержку
RELOAD — Обновляет папку Scripts

Скрипты для новичков 📄

В Sphire Executor есть .txt файл с простыми скриптами для начинающих.

Вы можете открыть его через кнопку OPEN и сразу протестировать.

Как добавить свои скрипты? 📄

Создайте или скачайте файл скрипта (.txt или .lua)
Поместите его в папку Scripts (рядом с .exe файлом)
Если папки нет — создайте её вручную
Убедитесь, что файл содержит корректный Lua-код

Возможные проблемы ❗

Executor не подключается
Убедитесь, что Roblox запущен перед нажатием INJECT

Игра зависает или вылетает
Скрипт может быть устаревшим или сломанным

Предупреждения ⚠️

Использование executor может нарушать правила Roblox
Вы можете получить бан

Как оставаться в безопасности 🌌

Не используйте executor на основном аккаунте
Не запускайте подозрительные скрипты
Старайтесь не использовать читы слишком явно
Не рассказывайте другим, что используете executor
